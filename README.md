# Portafolio
Portafolio de egresada de Ingeniería en Informática, mostrando proyectos de desarrollo web y diseño de interfaces con énfasis en claridad, modularidad y experiencia de usuario.
Este es mi portafolio personal desarrollado con **React** y **styled-components**, mostrando proyectos de frontend, UI/UX y diseño de interfaces.

---

## Requisitos previos

Antes de comenzar, asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión 16 o superior recomendada)
- [npm](https://www.npmjs.com/) como gestor de paquetes
- Git para clonar el repositorio

---
##Notas

-El formulario de contacto está conectado a Formspree (https://formspree.io/f/xreaadqb) para envío de correos.
- No necesitas configurar servidor propio: todo funciona con el endpoint gratuito de Formspree.
- Los estilos están hechos con Bootstrap y styled-components para modularidad y reutilización.
